
#Primeiro Projeto - PG

##Informacoes do Projeto

Tema do Projeto: Animação de uma Imagem através de Operadores Afins Bidimensionais 

Dupla: -Miriane Silva Trajando do Nascimento
       -Ullayne Fernandes Farias de Lima

Neste projeto e utilizado a biblioteca opencv para o tratamento da imagem e a IDE QT para fazer a inferface grafica

##Para compilar o programa

1. Instalar a biblioteca opencv no pc 
2. Instalar a IDE QTCreator
3. Linkar a biblioteca opencv com a IDE

OBS: Rodar o programa no ubuntu, versao utilizada para teste foi 16.10
